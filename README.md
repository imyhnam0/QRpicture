앱 플로우

홈 화면 → [새로 만들기]
→ Step 1: 사진 선택 (폴라로이드 프리뷰 포함)
→ Step 2: 음성 녹음 (미리듣기 / 재녹음 가능)
→ Step 3: 업로드 처리 (진행률 표시)
→ 미리보기 화면 → 갤러리 저장

  ---
실행 전 필수 설정 3가지

1. Firebase Hosting 배포 (웹 플레이어)

firebase deploy --only hosting

2. Firestore 규칙 배포

firebase deploy --only firestore:rules

3. Storage 규칙 배포

firebase deploy --only storage

iOS Podfile 최소 버전 확인

record 패키지가 iOS 12.0+ 요구. Podfile에 아래 줄이 있는지 확인:
platform :ios, '12.0'

Android minSdk

record 패키지가 API 21+ 요구 (Flutter 기본값이 21이라 별도 변경 불필요).
